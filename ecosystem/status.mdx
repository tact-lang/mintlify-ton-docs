title: "Network status"
---

This page lists websites that show if specific parts of TON blockchain are working normally.

|                                                       |                                                               |
|-------------------------------------------------------|---------------------------------------------------------------|
| [tonstat.us](https://tonstat.us/)                     | HTTP and ADNL server availability and performance.            |
| [status.toncenter.com](https://status.toncenter.com/) | Low-level metrics, such as latencies, rates, and loads.       |
| [validators.ton.org](https://validators.ton.org/)     | Official validation dashboard.                                |
| [tonscan.com](https://tonscan.com/validation)         | Pretty validation dashboard.                                  |
| [t.me/tonstatus](https://t.me/tonstatus)              | Notifications and requests for action for mainnet validators. |
| [t.me/testnetstatus](https://t.me/testnetstatus)      | Notifications and requests for action for testnet validators. |
| [t.me/validators](https://t.me/validators)            | Bot for validator owners to track their status.               |
