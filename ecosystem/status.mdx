---
title: "Network status"
---

Websites that show if specific parts of The Open Network (TON) blockchain are working normally.

| Resource                       | What it shows                                                 |
|--------------------------------|---------------------------------------------------------------|
| [tonstat.us](https://tonstat.us/)            | Hypertext Transfer Protocol (HTTP) and Abstract Datagram Network Layer (ADNL) server availability and performance. |
| [status.toncenter.com](https://status.toncenter.com/)  | Low-level metrics, such as latencies, rates, and loads.       |
| [validators.ton.org](https://validators.ton.org/)    | Official validation dashboard.                                |
| [tonscan.com/validation](https://tonscan.com/validation) | Validation dashboard.                                  |
| [@tonstatus](https://t.me/tonstatus)         | Notifications and action requests for mainnet validators. |
| [@testnetstatus](https://t.me/testnetstatus)     | Notifications and action requests for testnet validators. |
| [@validators](https://t.me/validators)        | Bot for validator operators to track their status.               |
