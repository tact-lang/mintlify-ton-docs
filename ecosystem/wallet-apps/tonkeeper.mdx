# Using Tonkeeper

## Overview

Tonkeeper is a central wallet in the TON Ecosystem. The Tonkeeper team developed it and has active support from both users and developers.

<img src="/img/docs/wallet-apps/Tonkeeper.png?raw=true" alt="Tonkeeper" width="240" height="480" style={{ display: 'block', margin: 'auto' }}/>

## Highlights

- Most widely used wallet in the TON ecosystem with millions of active users
- Supports TON coins, jettons (tokens), and NFTs
- Integrated dApp browser with TON Connect support
- Built-in token swap and staking features
- Available on iOS, Android, Chrome, Firefox, and desktop (Windows, macOS, Linux)
- Supports W5 wallet standard with 25% lower fees
- Battery feature for gasless transactions

## Creating a Wallet

1. Download and open Tonkeeper from the official sources ([tonkeeper.com](https://tonkeeper.com))
2. Select **Create new wallet**
3. Set a secure PIN code
4. **Write down your 24-word recovery phrase** and store it safely
   - This phrase is the only way to recover your wallet
   - Never share it with anyone
5. Verify the recovery phrase by selecting the correct words
6. (Optional) Enable biometric authentication for quicker access

Your wallet is now ready to use!

## Transferring Coins

### Sending TON

1. Open Tonkeeper and tap **Send**
2. Enter the recipient's address or scan their QR code
   - Accepts TON addresses or TON DNS names (username.ton)
3. Enter the amount to send
4. (Optional) Add a comment/memo
5. Review the transaction details and fee (~0.0055 TON)
6. Confirm with your PIN or biometric authentication
7. Transaction confirms in 5-10 seconds

### Receiving TON

1. Tap **Receive** on the main screen
2. Share your wallet address or QR code with the sender
3. Incoming transactions appear automatically once confirmed

## Switching to Testnet

### Mobile App (iOS/Android)

1. Open Tonkeeper and go to **Settings**
2. Scroll to the bottom and **tap the Tonkeeper icon 5-7 times rapidly**
3. The Developer Menu appears
4. Enable **"Switch to Testnet"** or **"Dev Mode"**
5. Return to the main screen
6. Tap your wallet name at the top
7. Select **Add Wallet** â†’ **Testnet Account**
8. Enter your 24-word recovery phrase from your mainnet wallet
   - The testnet wallet uses the same phrase but generates different addresses

Your testnet wallet is now active!

### Browser Extension

1. Open Tonkeeper extension
2. Go to **Settings**
3. **Tap the Tonkeeper logo 5-6 times rapidly**
4. Enable testnet mode when the Developer Menu appears
5. Follow steps 6-8 from the mobile instructions above

**Note:** If you encounter a blank screen, try using the mobile app instead or contact [@tonkeeper_supportbot](https://t.me/tonkeeper_supportbot).

## Managing Multiple Wallets

1. Tap your wallet name at the top of the main screen
2. Select **Add Wallet**
3. Choose to create a new wallet or import an existing one
4. Each wallet has its own 24-word recovery phrase
