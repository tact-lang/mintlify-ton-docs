[TVM Explorer](https://tvm-explorer.netlify.app/) is an interactive tool for exploring TVM assembly.\
Use it to see how your [Fift](/language/fift) code compiles to TVM assembly.

![TVM Explorer interface](/resources/images/tvmexplorer.png)

## What it does

- **Interactive execution** — Examine TVM bitcode and understand instruction behavior
- **Compilation visualization** — See how high-level code translates to TVM instructions

## Use cases

- Learn how different code patterns compile to TVM bitcode
- Debug compilation issues by examining bitcode
- Understand TVM instruction behavior
- Educational tool for TVM architecture
