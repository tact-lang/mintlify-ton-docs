To display an image, use the `<Image>` component. It allows displaying either:

- A single image for all themes (`src`)
  - When an SVG image is specified and there is no `darkSrc` alternative, its colors will be inverted in the dark mode
- One image per light theme (`src`) and one image per dark theme (`darkSrc`)
  - You can optionally specify `alt`  and `darkAlt`

Additionally, you can specify the `href` property, making an image double as a clickable link. When doing so, the `target` property allows to change the browsing context of the link: `_self` (default), `_blank`, or [other standard keywords](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/a#target).

Notice that making an image into a clickable link disables image zoom capabilities.

## Import

```tsx
import { Image } from '/snippets/image.jsx';
```

## Usage

Show images using the `<Image>` component.

```mdx
import { Image } from '/snippets/image.jsx';

{/* A single image */}
<Image src="/resources/images/idk.jpg"/>

{/* Image variations for light or dark themes */}
<Image
  src="/resources/images/<IMAGE>"
  darkSrc="/resources/images/<IMAGE_DARK_THEME>"
  />

{/* Image with all properties set */}
<Image
  src="/resources/images/<IMAGE>"
  alt="<ALT_TEXT>"
  darkSrc="/resources/images/<IMAGE_DARK_THEME>"
  darkAlt="<ALT_TEXT_DARK_THEME>"
  href="<CLICKABLE_LINK>"
  target="_blank"
  />
```
