import { Image } from '/snippets/image.jsx';

To display an image, use the `<Aside>` component.

## Import

```tsx
import { Image } from '/snippets/image.jsx';
```

## Usage

```mdx
import { Image } from '/snippets/image.jsx';

<Image src="/resources/images/idk.jpg"/>

<Image
  src="/resources/images/idk.jpg"
  darkSrc="/resources/images/idk-dark.jpg"
  />

<Image
  src="/resources/images/idk.jpg"
  alt="Something"
  darkSrc="/resources/images/idk-dark.jpg"
  href="LINK!"
  target="_blank"
  />
```
